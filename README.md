# Vhumatshelo Skills Development — Website

## How to Run Locally

### Requirements
- Node.js (download from https://nodejs.org — get the LTS version)

### Steps

1. Open this folder in VS Code
2. Open the terminal: press Ctrl + ` (backtick)
3. Type this and press Enter:

   npm install

4. Once that finishes, type:

   npm run dev

5. Open your browser and go to: http://localhost:5173

That's it! The website will open.

### To stop the server
Press Ctrl + C in the terminal.

### To restart
Run `npm run dev` again.
